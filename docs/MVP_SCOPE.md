# MVP Scope

The MVP is the smallest useful version of ChatCRM.

## Build First

- Dashboard with lead counts
- Leads table
- Add lead form
- Edit lead form
- Delete lead action
- Pipeline status
- Lead detail page
- Notes per lead
- Basic search

## Save For Later

- AI parsing
- Real SMS/calling
- Real map integrations
- Complex permissions
- Paid billing
- Advanced analytics

## Why Start Small

The blueprint is strong, but it is too large to build all at once. Starting with the core CRM makes the app easier to test, easier to understand, and easier to expand.
